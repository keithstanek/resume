Stanek Token Exchange

My First Decentralized Application - September 2021
I created this project to learn how to develop a Token Exchange UI that integrates with smart contracts. I completed the UI by going through the DAPP University Blockchain Mastery class. With solid programming foundation, I found the concepts of blockchain development easy to grasp and enjoyed all the challenges that came with learning the fundamentals. The UI is created using the React framework deployed on an apache server with my hosting company. The smart contracts were developed locally and deployed using Ganache, then migrated to the Rinkeby Network to be accessed publicly if you want to play around with the UI.
To use the exchange, please use either a chrome or brave browser, and make sure you have MetaMask installed. If you need some test ether to deposit into the exchange and buy my tokens, you can get some here. When on MetaMask, please make sure you are connected to the Rinkeby Network. The Token smart contract can be viewed on EtherScan.io here. The Exchange smart contract can be viewed on EtherScan.io here. If you want direct access to the source code, you can view it on my github page here.
